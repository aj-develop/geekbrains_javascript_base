Почему код даёт именно такие результаты?
var a = 1, b = 1, c, d;

1) c = ++a; alert(c);           // 2
В префиксной форме инкрементирование производится сразу,
а возврат — уже с обновленным значением.

2) d = b++; alert(d);           // 1
В постфиксной форме сначала происходит возвращение значения,
а потом выполняется инкрементирование.

3) c = (2+ ++a); alert(c);      // 5
В первом шаге "а" было увеличено до 2.
В этом шаге до 3.
2+3=5

4) d = (2+ b++); alert(d);      // 4
На втором шаге "b" было увеличено до 2 и это значение было сложенно с 2.
Потом  "b" было увеличено до 3.

5) alert(a);                    // 3
В третьем шаге "а" было увеличено до 3.

6) alert(b);                    // 3
На 4 шаге "b" было увеличено до 3.
